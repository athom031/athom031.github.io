User Engagement Map - Full Stack Web Application

July 2020

Project Pipeline: Backend - Node.js, Frontend - React, Other - MongoDB, Google Cloud APIs, Postman


A visual map of users for a client to see user whereabouts and activity. User side involves sleek Login and Registration tabs that then stores information in a MongoDB collection. Once logged in Users can change their information as well. With the Google Cloud API, the users coordinates are saved and displayed on the client side map. Both client and user side call on backend to view user activity in real time.


Demo


Interface to register new users:

  • Users could sign up using their personal information (username, name, password, U.S. Address)
  • Data validation is in place to ensure information provided meets constraints
  • User data is encrypted to ensure security before transmitting over servers to MongoDB


User Homepage for Logged In Users:

  • Users can log into their previously registerd accounts
  • Users are able to edit their information while logged in, seeing their changes update in real time
  • Data is stored in MongoDB collection maintaining data security from user registration


Client Side Product:

  • Google Cloud APIs integrate with MongoDB data to display user locations on a map
  • Features real-time updates of user activity and login status
  • Clients are able to see public information of users but not private data
